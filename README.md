# HentaiOS - hOS! - Rika #

### Sync ###

```bash

# Initialize local repository
repo init -u https://github.com/hentaiOS-legacy/platform_manifest -b Rika

# Sync
repo sync -c -j$(nproc --all) --force-sync --no-clone-bundle --no-tags
```

### Build ###

```bash

# Set up environment
$ . build/envsetup.sh

# Choose a target
$ lunch hentai_$device-userdebug

# Build the code
$ make otapackage -jX
```
